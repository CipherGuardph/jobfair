# Job Fair Queue System

Firebase-based paperless job fair recruitment and queueing app built with React, Vite, Firestore, Storage, Auth, and Cloud Functions.

## Setup

1. Copy `.env.example` to `.env` and fill in your Firebase web app config.
2. Install dependencies in the root and in `functions/`.
3. Run the Vite app with `npm run dev`.
4. Deploy Firestore rules, Storage rules, Hosting, and Functions through Firebase CLI.

## Included

- HR authentication
- Job fair creation
- Public applicant submission flow
- Resume upload
- Queue number generation
- Applicant dashboard and detail editing
- CSV export helper
- Security rules and callable functions scaffolding
